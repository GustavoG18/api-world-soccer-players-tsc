Este proyecto sera una API REST (Siguiendo fielmente a los principios de API REST) para practicar back-end con Node.JS y TypeScript en la cual esencialmente devolver los jugadores que participaron en un mundial x.
Por ejemplo, devolvera los equipos que participaron en el mundial 2006 con sus respectivos grupos asignados y un mapa de como se desarrollo ese mundial.

Peticiones GET
- Obtener los paises que participaron en X mundial.
- Obtener los paises que participarion en X mundial y sus fases de grupos.
- Obtener el mapa de los resultados de dicho mundial.
